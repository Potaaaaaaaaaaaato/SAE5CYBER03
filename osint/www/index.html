À propos de nous

Entreprise Fictive est une PME dynamique spécialisée dans les services informatiques, avec un focus sur la sécurité et la gestion des systèmes. Nous utilisons des technologies modernes comme un Active Directory pour l’authentification et un VPN pour le télétravail, garantissant une productivité optimale pour nos collaborateurs. Note : Notre VPN utilise un certificat auto-signé pour les tests internes (à remplacer bientôt).

Notre équipe, menée par Bob Martin (administrateur IT), et Alice Dupont (stagiaire dédiée à la maintenance), travaille dur pour maintenir nos infrastructures à jour. Par exemple, notre base de données MySQL est utilisée pour stocker des infos utilisateurs, mais des patchs pour CVE-2021-XXXX sont en cours. Découvrez notre équipe et nos dernières actualités !

Nos services

  • Gestion de réseaux sécurisés (pare-feu, VLANs) – Notre switch Cisco utilise SNMP public pour monitoring temporaire (communauté "public").
  • Authentification avancée avec Active Directory – Non patché contre Zerologon pour l'instant, tests en cours par Alice.
  • Télétravail via VPN SSL – Adresse 203.0.113.10:443, creds temporaires disponibles via Bob.
  • Stockage et partage de données (Samba) – Port 445 ouvert pour tests internes, vulnérable à EternalBlue si non patché.

Nous sommes en pleine migration vers des outils plus robustes, mais certains services comme notre base de données interne sont encore en test avec auth faible.
